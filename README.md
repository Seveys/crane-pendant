# Crane Pendant Station Configurator

A React-based web application that helps users quickly and accurately configure pendant stations for cranes. The tool outputs part numbers, recommended component sets, replacement options, or full pre-configured pendant assemblies based on user requirements.

This project was developed to:
- Reduce unnecessary vendor support requests
- Increase customer response time and accuracy
- Simplify pendant station configuration and ordering
- Provide a free, open, and reliable tool for crane equipment professionals

## Features
- Interactive Configuration Wizard
- Automatic Part Number Generation
- Preconfigured Pendant Assemblies
- Replacement Part Lookup
- Search & Filter Tools
- No Backend Required
- Fast, Error-Free Matching Engine

## Installation

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/yourusername/crane-pendant-configurator.git](https://github.com/yourusername/crane-pendant-configurator.git)
   cd crane-pendant-configurator
2. **Clone the repository:**
Install dependencies:

Bash

npm install


2. **Start the development server:**

Bash

npm run dev
Open your browser and navigate to the URL shown in the terminal (usually http://localhost:5173).

Usage
Launch the app using npm run dev.

Choose:

Full pendant configuration,

Replacement parts, or

A preconfigured station.

Enter pendant requirements (motions, speed, accessories, E-stop, etc.)

Receive:

Part numbers

Assembly list

Replacement recommendations

Notes / diagrams (if enabled)

Why This Tool Exists
Vendor support teams often receive basic pendant part-number inquiries that waste valuable time and disrupt workflow. This tool eliminates repetitive requests and gives users immediate, accurate answers.
